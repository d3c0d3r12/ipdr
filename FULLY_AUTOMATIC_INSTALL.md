## 🔧 **WHAT IT DOES:**

### **1. Downloads Python:**
```
URL: https://www.python.org/ftp/python/3.11.7/python-3.11.7-amd64.exe
Size: ~30 MB
Saves to: temp_installers/python-installer.exe
```

### **2. Installs Python:**
```
Command: python-installer.exe /quiet InstallAllUsers=1 PrependPath=1
Result: Python installed to C:\Program Files\Python311
PATH: Automatically added
```

### **3. Downloads Node.js:**
```
URL: https://nodejs.org/dist/v20.10.0/node-v20.10.0-x64.msi
Size: ~50 MB
Saves to: temp_installers/nodejs-installer.msi
```

### **4. Installs Node.js:**
```
Command: msiexec /i nodejs-installer.msi /quiet /norestart
Result: Node.js installed to C:\Program Files\nodejs
PATH: Automatically added
```

### **5. Installs Packages:**
```
Backend: pip install -r requirements.txt
Frontend: npm install
```

### **6. Cleanup:**
```
Deletes: temp_installers/ folder
Removes: Downloaded installers
```
