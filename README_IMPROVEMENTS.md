## 🎬 **VIDEO BACKGROUND FEATURE:**

### **How It Works:**
1. Auto-detects if video file exists
2. Loads video in background
3. Applies blur and opacity effects
4. GPU-accelerated rendering
5. No performance impact

### **How to Enable:**
```bash
# Step 1: Get a video
# Visit: https://www.pexels.com/videos/
# Search: "cyber security background" or "technology"

# Step 2: Add to project
# Rename to: background-video.mp4
# Place in: frontend/public/background-video.mp4

# Step 3: Restart app
# Video will auto-load!
```

### **Recommended Videos:**
- **Cyber Security:** Matrix code, digital networks
- **Law Enforcement:** Command centers, surveillance
- **Technical:** Server rooms, data streams
- **Abstract:** Particle effects, light trails

### **Video Specs:**
- Format: MP4 (H.264)
- Resolution: 1920x1080 or 1280x720
- File Size: < 10MB (compressed)
- Duration: 10-30 seconds (loops)
