# IPDR Tracking Hub

This project provides a secure, real-time IP Data Record (IPDR) tracing and enrichment system for Delhi Police Cyber Cell investigators. See `docs/requirements.md` for scope and constraints.

## Status
Planning phase. Next steps: confirm scope, choose stack, initialize repo.

## Getting Started (planned)
- Windows and Linux supported via Docker Compose
- API + Web + Worker services
- Postgres, Redis, and object storage
