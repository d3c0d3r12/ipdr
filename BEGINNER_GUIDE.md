## 🚀 **2 SUPER SIMPLE OPTIONS:**

**Just 1 Click - Everything Automatic!**

**Double-click:** `AUTO-INSTALL.bat`

This will automatically:
- ✅ Download Python 3.11

- ✅ Install both with PATH
- ✅ Install all packages
- ✅ Everything done in 30 minutes!

**NO MANUAL STEPS!** Just click and wait! 🎉

---

### **OPTION 2: Manual Installation** (If you prefer)

### **Step 1: Install Prerequisites** (One-Time, 10 minutes)

**Install Python:**
1. Go to https://python.org
2. Download Python 3.8 or higher
3. Run installer
4. ⚠️ **CHECK "Add Python to PATH"** (IMPORTANT!)
5. Click "Install Now"

**Install Node.js:**
1. Go to https://nodejs.org
2. Download LTS version
3. Run installer
4. Click "Next" through all steps

### **Step 2: Install Dependencies** (One-Time, 15 minutes)

**Double-click:** `INSTALL.bat` (in main folder)

Wait for it to finish. You'll see:
```
✅ Python packages installed
✅ Node.js packages installed
✅ Installation complete!
```

---

### **Step 3: Start Application** (Daily, 1 minute)

**Double-click:** `START.bat` (in main folder)

Browser opens automatically to http://localhost:3000

## 🔧 **TROUBLESHOOTING:**

### **"Python is not installed"**
- Install Python from python.org
- ⚠️ Must check "Add to PATH"
- Restart computer
- Try INSTALL.bat again

### **"Node.js is not installed"**
- Install Node.js from nodejs.org
- Restart computer
- Try INSTALL.bat again

### **Servers won't start**
- Double-click STOP.bat first
- Then START.bat again
