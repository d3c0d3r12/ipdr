# 🔧 Setup Scripts Folder

This folder contains all the setup and maintenance scripts for the IPDR Tracking Hub.

## 📁 Files in This Folder:

### **Installation:**
- **install.bat** - Installs all dependencies (Python & Node.js packages)

### **Running:**
- **setup.bat** - Simple startup (starts both servers)
- **start-servers.bat** - Enhanced startup with checks

### **Stopping:**
- **stop-servers.bat** - Stops all running servers

## 🚀 How to Use:

### **From Project Root (Recommended):**

Use the simple launchers in the main folder:

```cmd
INSTALL.bat    # Install dependencies
START.bat      # Start application
STOP.bat       # Stop application
```

### **From This Folder:**

You can also run scripts directly from here:

```cmd
install.bat         # Install dependencies
setup.bat           # Start servers (simple)
start-servers.bat   # Start servers (with checks)
stop-servers.bat    # Stop servers
```

## 📋 First Time Setup:

1. Install Python & Node.js
2. Run `INSTALL.bat` from project root
3. Run `START.bat` from project root
4. Use the application!

## 💡 Tips:

- Keep this folder organized
- Don't modify script paths
- Use root launchers for simplicity
- Check documentation for details
