## 🔧 **TROUBLESHOOTING:**

### **Common Issues & Solutions:**

#### **1. "Python is not installed"**
```
Problem: install.bat shows Python not found
Solution:
  1. Install Python from python.org
  2. ✅ MUST check "Add Python to PATH"
  3. Restart computer
  4. Run install.bat again
```

#### **2. "Node.js is not installed"**
```
Problem: install.bat shows Node.js not found
Solution:
  1. Install Node.js from nodejs.org
  2. Choose LTS version
  3. Restart computer
  4. Run install.bat again
```

#### **3. "pip install failed"**
```
Problem: Backend dependencies fail to install
Solution:
  1. Check internet connection
  2. Update pip: python -m pip install --upgrade pip
  3. Try manual install: cd backend && pip install -r requirements.txt
  4. If specific package fails, install separately
```

#### **4. "npm install failed"**
```
Problem: Frontend dependencies fail to install
Solution:
  1. Check internet connection
  2. Clear npm cache: npm cache clean --force
  3. Delete node_modules folder
  4. Try again: cd frontend && npm install
```

#### **5. "Port already in use"**
```
Problem: Servers won't start
Solution:
  1. Run stop-servers.bat
  2. Close any existing server windows
  3. Check Task Manager for python.exe or node.exe
  4. Kill those processes
  5. Try starting again
```
