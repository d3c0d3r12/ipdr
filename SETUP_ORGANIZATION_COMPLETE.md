## 🚀 **HOW IT WORKS:**

### **For Beginners:**

**They only see 3 simple files:**

1. **INSTALL.bat** ⭐
   - Click this FIRST (one-time)
   - Installs all dependencies
   - Takes 10-15 minutes

2. **START.bat** 🚀
   - Click this DAILY
   - Starts the application
   - Opens browser automatically

3. **STOP.bat** 🛑
   - Click this to STOP
   - Stops all servers
   - Clean shutdown

### **Behind the Scenes:**

The launchers call the actual scripts in `setup/` folder:

```
INSTALL.bat → setup/install.bat
START.bat → setup/setup.bat
STOP.bat → setup/stop-servers.bat
```
