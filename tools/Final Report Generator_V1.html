Final Report Generator

User Guide:

  • Select the input file by clicking the "Choose File" button below.
  • Choose Configuration i.e., Jio, Vodafone Idea or Complete Data
  • Download the Final Report by clicking "Download Final Report".

Points to Remember:

  • The input file must be in ".csv" format
  • There should not be any header row in input file.
  • Please ensure that there is no comma `,` in and field's value.
  • Please review that input and output(downloaded) file contains same number of data rows.
Example of a row:
2025-01-04 17:10:52 Z,192.168.1.1,India,Meghalaya,Shillong,Reliance Jio
