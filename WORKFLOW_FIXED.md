## 🐛 **Troubleshooting:**

### **Issue: "Nothing happens after upload"**

**Solution:**
1. Check if "Bypass Cloudflare" is checked
2. If not, look for the green button "Start Unlimited IP Lookup"
3. Click the button to start

### **Issue: "Button doesn't appear"**

**Solution:**
1. Check browser console for errors
2. Verify `runDir` value is set
3. Refresh the page and try again

### **Issue: "Redirect doesn't work"**

**Solution:**
1. Check if `navigateTo` is available (Nuxt 3)
2. Try manual navigation: `/ip-lookup?run_dir=...&auto_start=true`
3. Check browser console for errors

### **Issue: "Terminal doesn't start"**

**Solution:**
1. Check if backend is running: `http://localhost:8000/docs`
2. Verify API endpoint: `/api/lookup/stream`
3. Check browser console for SSE errors
4. Verify run directory path is correct
