# 📦 Install UI Enhancement Packages

Run these commands in the frontend directory:

```bash
cd "C:\Users\saheb\Downloads\New FIR\frontend"

# Core UI packages
npm install @heroicons/vue@2.0.18
npm install lucide-vue-next@0.292.0

# Charts and visualizations
npm install chart.js@4.4.0 vue-chartjs@5.2.0
npm install apexcharts@3.44.0 vue3-apexcharts@1.4.4

# Map for geolocation
npm install leaflet@1.9.4
npm install @vue-leaflet/vue-leaflet@0.10.1

# Animations
npm install @vueuse/motion@2.0.0
npm install @vueuse/core@10.6.1

# Date utilities
npm install date-fns@2.30.0

# Notifications
npm install vue-toastification@2.0.0-rc.5
```

After installation, restart your dev server:
```bash
npm run dev
```
